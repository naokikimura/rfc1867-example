rfc1867-example
===============

Quick Start
-----------

    git clone https://github.com/naokikimura/rfc1867-example.git
    cd rfc1867-example
    mvn jetty:run

Requires
--------

- <abbr title="Java Development Kit">JDK</abbr> 7+
- [Apache Maven][2] 3+

See also
--------

- [RFC 1867][1]
- [Apache Maven][2]
